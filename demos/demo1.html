Introduction

  • This is a demo page showing the implementation of generating a list of related terms with a specified keyword.
  • This page assumes that hosting website include pdb-onsite js
  • The only resource hosting website needs is the onsite js

Features

  1. support customization of a tag
  2. support dynamic generation of related term list
  3. support rich HTML e.g. tooltip

Todos

  1. provide support on style customization
  2. provide css hooks for more customization
  3. manage dependencies. e.g. jQuery and Bootstrap in this case
